# Persistence

## Objective

Persistence techniques were intentionally excluded from Phase 7.

The purpose of this phase was to focus on:

- Threat Hunting
- Detection Engineering
- Lateral Movement
- Credential Validation
- Remote Execution

Persistence techniques will be implemented during later phases of the SOC Lab.

Examples include:

- Scheduled Tasks
- Registry Run Keys
- Startup Folder
- WMI Event Subscription
- Windows Services
- User Creation

These techniques will be documented in future phases.
