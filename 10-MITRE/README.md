# MITRE ATT&CK Mapping

This directory maps every attack simulation performed in the SOC Home Lab to the MITRE ATT&CK Enterprise Matrix.

Each phase documents:

- ATT&CK Tactic
- ATT&CK Technique
- Technique ID
- Description
- Detection Strategy
- Splunk Queries
- Evidence
- Phase Reference

The mapping is continuously updated as additional attack simulations are completed.
